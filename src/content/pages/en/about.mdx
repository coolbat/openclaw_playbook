## What this template is for

AnyPlaybook is a static Astro template for bilingual tutorial sites.

It is designed for sites that need:

- a homepage with a 5-step onboarding path
- a learn hub for deeper explanations
- a template library for reusable workflows
- a resources directory
- a symptom-first troubleshooting section

## What downstream sites should customize first

1. Site branding and domain config
2. Homepage messaging and CTAs
3. Content collections and slugs
4. Analytics, ads, and verification codes
