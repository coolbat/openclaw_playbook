## When to use this template

Use this when weekly reporting is still necessary, but manual collection wastes too much time. It is a coordination template more than a writing template.

## Minimal setup

Choose one reporting rhythm and one input pattern. If every teammate reports in a different format, normalize that first before expecting good automation.

## What good output looks like

The final update should make it easy to see wins, blockers, and next priorities. It should reduce manager follow-up, not create more of it.

## First extension

Add a short trend section that compares this week against the last one. That turns raw status into something more useful for planning.
